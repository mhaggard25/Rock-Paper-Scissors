# Rock Paper Scissors — Pygame Edition

A simple Rock–Paper–Scissors game written in **Python** using **Pygame**.  
This is an early build focused on getting the core game loop and logic working.

## 🎮 Current Features
- Click to choose **Rock**, **Paper**, or **Scissors**
- Computer randomly selects its choice after the player picks
- Win/loss/tie result is printed to the console
- Basic UI layout with image-based cards

## 🧠 What This Project Demonstrates
- Event-driven input handling in Pygame  
- Rendering images and colored selection boxes  
- Basic game-state management (round start, player choice, computer choice)  
- Simple turn-based logic inside a real-time game loop  

## 🚧 Future Plans
- On-screen score tracking  
- “Play Again” button or round reset animation  
- Sound effects for selections and results  
- Improved card graphics and UI polish  
- Fade-in/out or slide animations for results
